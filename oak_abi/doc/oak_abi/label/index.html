[−][src]Module oak_abi::label

Labels represent the kinds of information that is allowed to be processed by entities in the Oak system. The Oak Runtime allows Oak Nodes to manipulate labels, and so the labels need to be passed across the Oak ABI in a defined binary format. That format is a serialized protocol buffer holding the Label message defined in the policy.proto file.
